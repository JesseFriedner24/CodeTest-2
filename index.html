Questionnaire
How do you define an AngularJS application? An application built using the AngularJS framework, Angular is basically a library written in JavaScript.
What is a controller? A controller controls the data of Angular applications, we create contollers in JS files. A controller may have two methods: Functions, and Variables.
What does ng-model do? ng-model Binds input from the HTML to variables created in Javascript.
What is interpolation markup Interpolation markup is used by Angular for data-binding to text nodes and attribute values.
What is $scope? $scope Is the link between your view and controller. When you write {{name}}, Angular searches for name property in the scope.
What is dependency injection? You can use dependency injection when defining components such as Services and Directives, you can inject specific components with dependencies.
What is two-way binding? Two-way binding is when a model variable is bound to a HTML element that can change and display the value of the variable.
What is "controller as" syntax? Using controller as makes it clear which controller you are working with in the template when multiple controllers apply to an element.
What is MV* architecture? Model-View architecture is when the model responds to the request from view. The model manages application data, the view is a presentation of data.
What is the difference between a service and a factory? A factory gives us the function's return value, you can create an object, add properties to it, then return that same object. Services are used to communicate between controllers to share data you add properties to a service object by using the word "this".
